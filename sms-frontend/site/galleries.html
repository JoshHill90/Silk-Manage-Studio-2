Settings

Options to display the gallery on an external site.

External Access:

Switch the toggle to the 'on' position to enable external API calls for accessing this gallery and its data. This functionality can be utilized to serve images on your WordPress, Squarespace, or custom-built website.

External List Access:

Toggle the switch to the 'on' position to activate the display of this gallery through an external API call that requests a list of galleries

Share

Generate a link that you can share with external users to view the gallery

Random Order:

Toggle the switch to "on" to display the gallery in random order. This option affects only images on this shared gallery site.

Allow Downloads:

Toggle the switch to "on" to allow the option to download the images.

Set Views:

Image Gallery: This view presents images in a visually appealing bento box layout, featuring a header image at the top if available.

Image Review: In this view, images are displayed in a grid layout. You have the option to retain selected images and provide feedback through comments for review purposes.

Delete

This will only delete the gallery, and any shared gallery links. But not the images associated with it.
